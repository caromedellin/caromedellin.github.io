What does it mean to display inline vs using inline block? Every element is a box model on CSS it all has height and width. They all naturally display in 2 ways, B • Block boxes: the are created with elements like P DIV and table, they take up all of the width of the HTML page • Inline boxes: This ones are generated by tags live B I or span, and things like text and images. They coexist on the same line by they don't accept all the changes in sizes. By setting the display to inline or block or inli-block you can change the deffault value of the element Block Elements: Inline Elements: They don't break the flow of the document! Display values: basic ones ○ Inline ○ Block ○ Inline-block ○ None Some resources: There's more to this https://developer.mozilla.org/en-US/docs/Web/CSS/display
